# Projet Web Dev M1

## Alexandre Deprez, Théo Porzio, Loick Zirotti, Emile Cornu, Joaquim Dieu

### Prérequis

- NodeJS
- npm
- SQLite3

### Installation

- Cloner le projet
- `npm install`
- `npm run site:dev`
- `npm run api:start`

### Utilisation

- Aller sur `localhost:3000`
- Visiter le site comme bon vous semble

### Documentation API
